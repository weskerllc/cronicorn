================================================================================
KEY VALUE PROPOSITIONS
================================================================================

1. ZERO CODE CHANGES
   - No SDK to install or deploy
   - Just point at existing HTTP endpoints
   - 5-minute setup

2. AUTOMATIC ADAPTATION
   - AI learns from endpoint behavior
   - Tightens during failures
   - Relaxes during stability
   - No manual tuning required

3. SAFETY-FIRST DESIGN
   - Min/max constraints prevent runaway behavior
   - AI always respects boundaries
   - Graceful degradation (baseline continues if AI unavailable)

4. COMPLETE TRANSPARENCY
   - Every execution logged
   - Every AI decision explained in plain English
   - Full debugging context

5. DEVELOPER-FRIENDLY
   - Clean, intuitive UI
   - MCP integration for AI assistants
   - Self-hosting option available
   - Production-ready reliability

PRODUCTION REQUIREMENTS
================================================================================

Recording:
- Resolution: 1920x1080 @ 30fps minimum
- Software: OBS Studio, Loom, or ScreenFlow
- Browser: Clean profile, 100-125% zoom
- Environment: Quiet, Do Not Disturb enabled

Voiceover:
- Microphone: USB quality (Blue Yeti, Rode NT-USB)
- Tone: Professional but friendly
- Pace: 150 words/min (slow for concepts)
- Recording: Separate from screen (easier to edit)

Editing:
- Software: Premiere Pro, Final Cut, or DaVinci Resolve
- Transitions: Subtle crossfades (0.5-1s)
- Music: Optional, very low volume (-25 to -30 dB)
- Text: Large, readable at 720p

Export:
- Master: MP4 H.264 1080p @ 10-15 Mbps
- Web: MP4 H.264 1080p @ 5-8 Mbps
- Mobile: MP4 H.264 720p @ 2-4 Mbps
- GIF: 800x600 @ 15fps, 10 seconds (AI moment)
